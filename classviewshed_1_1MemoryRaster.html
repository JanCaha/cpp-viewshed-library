Detailed Description

Class that creates raster with single band purely in memory. The overall representation is very simplistic, basically a two dimensional array with some properties, that allow creating real QgsRasterLayer representation from this object.

Parameters
rasterTemplateThe memory representation gets number of columns and rows, coordinate system, extent and no data value based on this QgsRasterLayer.
dataTypetype of data to be stored. Default Qgis::DataType::Float32.
defaultValuedefault value to prefill the object with. Default is std::numeric_limits<double>::quiet_NaN().
